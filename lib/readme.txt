Readme Libraries of SemDwhSearch
--------------------------------

Library                                        What                                    Use

- commons-dbcp-1.4.jar                         Apache Commons DB connection pool       Data access
- commons-fileupload-1.2.1.jar                 Apache Commons File Upload              POST request
- commons-io-2.0.1.jar                         Apache Commons IO                       Used by Commons File Upload
- commons-lang-2.6.jar                         Apache Commons Lang                     Used for DBpedia extraction
- commons-pool-1.5.4.jar                       Apache Commons Pool                     Used by Commons DBCP
- jdom-1.1.1.jar                               JDOM                                    Reading and writing XML files
- junit-4.5.jar                                JUnit                                   Testing
- log4j-1.2.15.jar                             Log4j                                   Logging
- servlet-api-2.5-6.1.21.jar                   Java Servlet API (from Jetty 6.1.21)    Web frontend
- slf4j-api-1.5.8.jar                          SLF4j                                   Logging
- slf4j-log4j12-1.5.8.jar                      SLF4j binding to Log4j 1.2              Logging

jetty/:
- jetty-6.1.21.jar                             Jetty                                   Web frontend (standalone)
- jetty-util-6.1.21.jar                        Jetty Utilities                         Web frontend (standalone)

mysql/:
- mysql-connector-java-5.1.10.jar              MySQL JDBC Connector                    Database
